# Чистый лист — сброс окружения

Инструкция, чтобы убрать всё установленное и развёрнутое (локально и на сервере) и начать с нуля.

## Локально

```bash
# Из корня проекта my_treds
rm -rf node_modules .next out data/*.db
# Пересоздать зависимости (когда будете снова поднимать проект)
npm install
npx prisma generate
```

Файл `.env` не трогаем — в репо его нет, он локальный. При необходимости создайте заново из `.env.example`.

## Сервер Beget (root@5.35.89.34)

Выполнить по SSH (ключ: `-i ~/.ssh/id_ed25519_beget`).

### 1. Остановить приложение и удалить проект

```bash
ssh -i ~/.ssh/id_ed25519_beget root@5.35.89.34

pm2 stop my_treds
pm2 delete my_treds
rm -rf /root/my_treds
```

### 2. Остановить Ollama (если была запущена)

```bash
pkill -f ollama || true
```

### 3. (Опционально) Удалить Ollama и модели

```bash
# Удалить сервис/симлинки, если настраивали
# Удалить данные Ollama (модели и т.д.)
rm -rf /usr/share/ollama /root/.ollama 2>/dev/null || true
# Если ставили через install.sh — полное удаление зависит от способа установки
```

После этого сервер чист от приложения my_treds и от Ollama. Для нового деплоя — клонировать репо заново и следовать `DEPLOY-BEGET.md`.
